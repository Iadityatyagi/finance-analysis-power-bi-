# 💳 Financial Transaction Analysis Dashboard

## 📌 Project Overview

Developed an interactive Power BI dashboard to analyze customer transaction data and generate actionable business insights. The project involved data cleaning, handling missing values, correcting data inconsistencies, data transformation, and creating KPI-driven visualizations.

## 🛠️ Data Preparation

* Cleaned and transformed raw transaction data.
* Handled missing and duplicate records.
* Standardized data formats and validated transaction records.
* Created calculated measures and KPIs using DAX.

## 📊 Key KPIs

* Total Transaction Amount with YoY Growth
* Total Transactions Count
* Average Transaction Value
* Total Fees Collected
* Total Tax Generated

## 📈 Dashboard Insights

### Dashboard 1 – Executive Summary

* Monthly Transaction Amount Trend Analysis
* Transaction Amount by Status (Success, Failed, Pending)
* Customer Segment Performance Analysis
* State-wise Transaction Analysis
* Transaction Type Profitability Analysis
* Gender-wise Transaction Contribution

### Dashboard 2 – Detailed Transaction Analysis

* Interactive drill-through functionality
* Record-level transaction details
* Advanced filtering and data exploration

## 🔍 Business Insights

* Identified top-performing customer segments and regions.
* Monitored transaction success rates and operational efficiency.
* Analyzed profitability through fees and tax generation.
* Tracked monthly transaction trends and customer behavior patterns.

## 🛠️ Tools Used

* Power BI
* Power Query
* DAX
* Excel
